Hiperlinks

Hiperlinks são ligações em hipertextos que ao serem clicados levaram o usuário para um certo destino, esses destinos podem ser:

  1. Externos
  2. Internos
  3. Arquivos
  4. Endereços de email

Marcação de Hiperlink

A marcação de hiperlinks é feito pela tag <a> e atributo href que referencia algum conteúdo.

Sintaxe

<a href="endereço">Conteúdo referenciado clicável</a>


Hiperlink Externo

Quando o hiperlink é externo, quer dizer que o hiperlink levará para outra página, onde o "endereço" será substituído pelo endereço URL da página de destino. Neste tipo é possível colocar o atributo target, que especifica se o link será aberto na própria pagina ou em outra guia.

Sintaxe

<a href="endereço URL" target= "_self" ou "_blank">Conteúdo referenciado clicável</a>
Onde "_self" levará na mesma pagina e "_blank" para uma nova guia.

Hiperlink Interno

No hiperlink interno, o hiperlink levará o usuário para uma parte em específico da página.

Sintaxe

Primeiro, é preciso dar um id ou nome para o destino, sendo assim:
<a name="nome">Conteúdo</a> ou id="nome", o id sendo um atributo para uma tag.
Depois de ter nomeado o destino, é apenas necessário utilizar <a href="#nome">Conteúdo referenciado clicável</a>

Hiperlink de Arquivo

Nesse tipo, o hiperlink permitirá o download de algum arquivo.

Sintaxe

<a href="pasta" download="arquivo" type="tipo">Conteúdo para Download</a>
Existem diversos tipos para esse hiperlink, o link para todos os tipos consta aqui: Common MIME Types

Hiperlink de Endereço Email

Nele, o hiperlink leva a um endereço email.

Sintaxe

<a href="mailto:email">email</a>
